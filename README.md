# Web Nalomy

Sitio web oficial de Nalomy, diseñado para la exhibición y gestión de experiencias artesanales de bienestar, incluyendo las exclusivas Mystic Box. Desarrollado con HTML, CSS y JS, este proyecto integra una estética cuidada con una estructura técnica sólida, reflejando el compromiso del emprendimiento con el autocuidado consciente.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
